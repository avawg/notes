
| 函数名| 处理 | 
|--- | --- |
| strcpy(dst, src)| 原字符串（包含尾部'\0'），写到目标字符串，可能越界 |
| strcat(dst, src)| 原字符串（包含尾部'\0'），追加写到目标字符串，可能越界 |
| sprintf(dst, "", src)| 将原字符串填充到模式串中，再写到目标字符串，可能越界 |
| strcpy_s(dst, n, src)| 确保目标字符串长度n能够容纳 原字符串（包含尾部'\0'），否则失败，不会占用非法空间|
| strcat_s(dst, n, src)| 确保目标字符串长度n能够容纳原字符串（包含尾部'\0'），否则失败，不会占用非法空间|
| strncpy(dst,src, n)| 从原字符串最多读n个字符，写到目标字符串，原字符串长度不足n用'\0'填充 |
| strncat(dst, src, n)| 从原字符串最多读n个字符，追写到目标字符串，末尾添上一个‘\0’, 原字符串长度不足n用'\0'填充  |
| strncpy_s(dst, m, src, n)| 从原字符串最多读n个字符，写到目标字符串，原字符串长度不足n用'\0'填充。确保目标字符串长度m能容纳n |
| strncat_s(dst, m, src, n)| 从原字符串最多读n个字符，写到目标字符串，末尾添上一个‘\0’, 原字符串长度不足n用'\0'填充 。确保目标字符串长度m能容纳n + 1  |

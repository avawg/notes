## 用户自定义变量

用户自定义变量 
```
var_name=value
```

变量调用
```
echo ${var_name}
```

查看系统所有变量
```
set
```

删除变量
```
unset var_name
```

ps：执行set -u后，echo ${var_name}系统不存在的变量，会提示不存在

## 环境变量  

export 将用户自定义变量导出到环境变量中

## 脚本执行方式

1.赋予执行权限，再运行。  
新建脚本文件，从网上下载文件，默认没有执行全线，因为安全。  
```
chmod 755 *.sh
*.sh
```
2.bash解释器执行脚本（不需要赋予权限）  
```
bash *.sh
```
